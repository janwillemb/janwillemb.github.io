Git Basics with Intellij

Initialization

Create a new Java project in Intellij (New> Next> Commandline App> Next> Enter a name> Finish). You can also open an existing project.

Choose "Enable Version Control Integration" in menu VCS.

Choose "Git" and OK. After a while Intellij reports that it was successful, bottom left:

Do your first commit now.

  1. Click on the "Commit" tab on the left
  2. Select all "Unversioned Files", these are the new files of your project that are not yet known in the git directory (in this case: all files, because the git repository is still empty)
  3. Enter a nice commit message that you will not regret later
  4. Press "Commit"

Git now asks for your name and email address. It doesn't matter what you fill in here: this is the name and email address with which your commit messages are "signed".

Intellij gives a message at the bottom that it was successful, with your own commit message:

Follow -up commits

Now make a change in your code, for example add this sensational text:

System.out.println ("Hello World!");

Save your change.

On the "commit" tab your changed file now appears. Make your second commit by checking it, filling in a commit message again and then press "Commit".

Your list of changes is now empty again.

View commit history

You can view all the commits that you have done, and differences between files, via the "log" tab of the "git window":

  1. Click on "git" at the bottom
  2. If necessary, click on "Log Master". You see a list of all commits in the middle window.
  3. Click on the commit that you want to view. You will see the details of the commit on the right.
  4. If you want to see what has been changed in a file, double-click that file.
